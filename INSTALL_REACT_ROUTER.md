# Install React Router

You need to install react-router-dom to use the React admin panel.

Run this command:
```bash
npm install react-router-dom
```

After installation, the admin panel will be available at:
- Login: `http://localhost:5173/admin/login` (or your dev server URL)
- Dashboard: `http://localhost:5173/admin/dashboard`

The React admin panel replaces the PHP admin pages.
